# 视觉设计规范（iOS · v0.1）

> 2026-06-24 · 配合 [线框图.md](线框图.md)。本期先覆盖「地图」「记录中」两页所需的全部样式。
> 落地：SwiftUI；颜色给 HEX，图标给 SF Symbols 名。

---

## 1. 设计调性

户外 / 清晰 / 可靠。地图为主角，UI 控件克制、半透明悬浮，**强对比保证强光下可读**（徒步常在户外强光）。主色用自然绿，强调户外属性，同时与两步路的绿色系一脉相承。

---

## 2. 配色

### 品牌色
| 名称 | HEX | 用途 |
|------|-----|------|
| Primary 主绿 | `#1F9D55` | 主 CTA（导航/开始记录）、选中态、强调 |
| Primary Dark | `#15803D` | 按压态、深色描边 |
| Primary Tint | `#E6F6EC` | 浅底（选中背景、标签底） |

### 功能色
| 名称 | HEX | 用途 |
|------|-----|------|
| Recording 红 | `#FF3B30` | 记录中指示点、结束按钮、实走轨迹 |
| Warning 橙 | `#F2730D` | 偏航横幅、警示 |
| Info 蓝 | `#2D7FF9` | 计划线（导航）、链接 |
| Contour 青 | `#36C5C0` | 等高线 |

### 中性色
| 名称 | HEX | 用途 |
|------|-----|------|
| Ink 主文字 | `#1C1C1E` | 卡片主文字 |
| Ink2 次文字 | `#6B7280` | 标签、次要信息 |
| Card 卡片 | `#FFFFFF` | 数据面板、弹窗 |
| Divider 分隔 | `#E5E7EB` | 线 |
| Scrim 蒙层 | `rgba(18,20,24,0.72)` | 地图上悬浮控件/信息条底色 |
| Scrim Light | `rgba(255,255,255,0.92)` | 浅色悬浮控件底 |

> 地图上的控件统一用 **Scrim 深底 + 白图标/白字**，或 **白底 + 深图标**，二选一保持一致；本规范地图控件用「白底圆钮 + 深图标 + 阴影」，信息条用「深 Scrim + 白字」。

## 3. 字体（SF Pro）

| 角色 | 字号/字重 | 说明 |
|------|-----------|------|
| 数据大数字 | 34 Bold（SF Pro Rounded） | 距离/用时等核心读数 |
| 数据中数字 | 22 Semibold（Rounded） | 海拔/爬升 |
| 标题 | 17 Semibold | 导航栏标题 |
| 正文 | 15 Regular | |
| 标签/单位 | 13 Medium，Ink2 | 数据标签、单位 |
| 信息条 | 12 Medium | 顶部坐标条 |

数字一律用 **SF Pro Rounded + 等宽数字（monospaced digits）**，跳动时不抖动。

## 4. 组件规范

| 组件 | 规格 |
|------|------|
| 圆形地图控件（定位/居中/图层…） | 44×44，白底 `#FFFFFF`，圆角=圆，阴影 `0 2 8 rgba(0,0,0,.18)`，图标 22 深灰 `#1C1C1E` |
| 缩放 +/- | 竖向胶囊，宽 44，两段，中间 1px 分隔；下接滑块轨道 |
| 缩放滑块 | 轨道宽 4，圆角；滑块 28 圆点，主绿 |
| 主 CTA 按钮 | 高 52，圆角 14，主绿底白字 17 Semibold；按压 Primary Dark |
| 次按钮（记录页暂停） | 高 52，圆角 14，白底 `#FFFFFF` + 深字 + 细描边 |
| 顶部信息条 | 高 ~36，Scrim 深底，白字 12，左对齐坐标/海拔/精度 |
| 数据面板卡 | 白底，圆角 20（仅顶部圆角，贴底），内边距 16，栅格 2 列 |
| 状态标签 | 圆角 8，12 Medium；记录中=红点+「记录中」，定位差=灰 |
| 偏航横幅 | 通栏，Warning 橙底白字，圆角 12，带震动 |

间距：基础 8 栅格（8/12/16/24）。安全区适配刘海/灵动岛与底部 Home 指示条。

## 5. 图标（SF Symbols 名）

| 功能 | SF Symbol |
|------|-----------|
| 定位/回到我的位置 | `location.fill` |
| 居中 | `scope` |
| 图层 | `square.3.stack.3d` |
| 叠加 | `square.2.layers.3d.fill` |
| 搜索 | `magnifyingglass` |
| 放大/缩小 | `plus` / `minus` |
| 公里标/里程碑 | `ruler` |
| 开始记录 | `record.circle` / `play.fill` |
| 导航 | `location.north.line.fill` |
| 暂停/继续/结束 | `pause.fill` / `play.fill` / `stop.fill` |
| 航点/标注 | `mappin.and.ellipse` |
| 海拔/爬升 | `arrow.up.right` / `mountain.2.fill` |
| 返回 | `chevron.left` |
