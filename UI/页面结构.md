# UI 页面结构（iOS · MVP）

> 版本：v0.1 · 2026-06-24
> 范围：MVP（仅 iOS，本地优先，无账号无云端）。参照两步路/六只脚的常见布局，按我们已定需求裁剪。
> 说明：账号/云端为三期，本结构里仅"我的"页预留入口位，不实现。

---

## 1. 信息架构（底部 Tab）

底部 **4 个 Tab**（从左到右）：

```
┌─────────────────────────────────────┐
│                                     │
│            (各页内容区)              │
│                                     │
├─────────────────────────────────────┤
│  🏠首页    🗺地图    📁轨迹    👤我的  │  ← 底部 Tab(4)
└─────────────────────────────────────┘
```

- **🏠 首页**（最左，默认进入）：累计数据卡 + 快捷入口（开始记录/离线地图）+ **活动·公告区**（运营发布的活动信息与公告，需联网拉取，MVP 作可选联网模块）。
- **🗺 地图**：**全屏沉浸地图，进入后隐藏底部 Tab**；记录与导航都从这里发起。
- **📁 轨迹**：本地轨迹库（列表）。
- **👤 我的**：设置、离线地图管理、导入、（三期）账号入口。

**Tab 显隐规则：**
- 显示 Tab：首页 / 轨迹 / 我的。
- **隐藏 Tab：地图（全屏）**；以及所有二级 push 页（详情/导航/记录/设置等）。

## 2. 页面清单与跳转关系

```
地图 Tab
 ├─ 地图主页（全屏地图）
 │    ├─▶ 轨迹记录页（开始记录后进入/叠加）
 │    ├─▶ 图层面板（右上「图层」）
 │    ├─▶ 叠加面板（右上「叠加」选多条轨迹）
 │    └─▶ 离线地图下载（框选区域下载）
 │
轨迹 Tab
 ├─ 轨迹库列表
 │    └─▶ 轨迹详情
 │          ├─▶ 沿轨迹导航页
 │          ├─▶ 导出/分享（系统分享面板）
 │          └─▶ 海拔剖面（页内）
 │
我的 Tab
 ├─ 我的（入口聚合）
 │    ├─▶ 设置（采样/偏航阈值/语音播报等）
 │    ├─▶ 离线地图管理（已下载区域）
 │    ├─▶ 导入轨迹（文件选择）
 │    └─▶ 账号登录（三期占位，MVP 灰显/隐藏）
 │
外部入口
 └─ 系统「用本应用打开」.gpx ─▶ 导入预览 ─▶ 存入轨迹库
```

## 3. 各页面元素

### 3.1 地图主页（首页）
- 全屏地图（MapLibre）。
- 右上：**【图层】**【**叠加】**按钮。
- 右侧中部：**缩放 +/-** + 滑块（上滑放大/下滑缩小）。
- 右侧下部：**公里标**按钮（叠加在轨迹上时生效）。
- 左下/右下：**定位（回到我的位置）**按钮、指南针。
- 底部中间：**【开始记录】**主按钮（醒目）。
- 入口：离线下载（可放在图层面板或我的页）。

### 3.2 轨迹记录页（记录中 / 已暂停）
- 地图 + 实时绘制的当前轨迹。
- **实时数据面板**：距离、用时、当前海拔、累计爬升/下降、（P1 速度/配速）、GPS 精度状态。
- 控制按钮：**暂停 / 继续 / 结束**；**打航点**（P1）。
- 顶部状态：记录中红点闪烁 / 已暂停 / 定位较差。
- 后台时：锁屏 Live Activity（P1）/ 蓝色定位状态条。
- 结束 → 结算保存弹窗（命名、备注、丢弃极短轨迹）。

### 3.3 沿轨迹导航页（核心交互页）
布局（对应离线地图模块第 7 节示意）：
```
┌───────────────────────────────┐
│                    [图层] [叠加]│
│                            [+] │
│                            [-] │
│            地图              ▌  │ ← 缩放滑块
│                          [公里标]│
│  [状态: 在线/偏航/定位差/已到达]  │
│  [剩余距离 · 剩余爬升]           │
└───────────────────────────────┘
```
- 计划线（醒目）+ 我的位置/朝向 + 起终点 + 沿线航点 + （开启则）实走线。
- 进入前：方向选择（正/反向）、是否同时记录开关。
- 偏航：横幅 + 震动 + （可选）语音播报。
- 海拔剖面（可展开/半屏）。

### 3.4 轨迹库列表（轨迹 Tab）
- 列表项：缩略图（地图小图）、名称、距离、爬升、日期、来源（记录/导入）。
- 排序/筛选（P1）；搜索（P1）。
- 右上：导入入口。
- 点击 → 轨迹详情。

### 3.5 轨迹详情
- 顶部地图（轨迹全貌）。
- **海拔剖面图**（点击联动地图）。
- 统计卡：距离、用时、累计爬升/下降、最高/最低海拔。
- 操作：**沿此轨迹徒步（导航）** / **导出** / **分享** / 重命名 / 删除。
- 航点列表（若有）。

### 3.6 离线地图下载 / 管理
- 下载：地图上**框选区域** + 选择最高级别（≤ z16）+ **预估体积** → 下载（进度）。
- 管理：已下载区域列表、占用空间、删除、更新。
- 等高线开关（是否随包下载/显示）。

### 3.7 导入预览
- 解析后：地图预览 + 统计 + 航点数。
- 容错提示（无海拔/无时间/多轨迹自动全导入）。
- 确认导入 → 轨迹库。

### 3.8 设置
- 记录：采样（默认 5s/5m）、自动暂停开关、海拔来源（气压计优先）。
- 导航：偏航阈值（默认 25m，可改）、语音播报开关与间隔（5/10 分钟）、同时记录默认开关。
- 通用：单位（公制）、关于、（三期）账号。

### 3.9 我的（入口聚合）
- 头部：（三期）登录入口 / MVP 显示"本地模式"。
- 列表入口：离线地图、导入轨迹、设置、关于。

## 4. 与两步路/六只脚的异同（设计取向）

- **同**：地图为中心、底部 Tab、轨迹库 + 详情 + 海拔图、记录浮层、导航偏航提醒。
- **异（MVP 做减法）**：无社区/广场/动态、无账号、无在线轨迹搜索；分享只走系统文件分享。
- 这些被砍的部分在三期随后端补齐。
