# 低保真线框图（iOS · MVP）

> 版本：v0.1 · 2026-06-24
> 依据：`UI/UI 参考/` 下 11 张两步路实拍截图 + 已定 MVP 需求。
> 约定：`[按钮]` 可点元素，`( )` 图标，`«MVP砍»` = 本期不做，`«P1»` = 二期。
> 目标：对齐两步路成熟布局，砍掉社区/云端/3D/游记等非 MVP 部分。

- 全屏沉浸，**不显示底部 Tab**；左上「返回」回到来源 Tab（如首页）。
- **信息条（坐标/海拔/定位误差）改为靠上、左右居中**。
- **去掉搜索（放大镜）图标**。
- **新增「工具」按钮** → 左侧抽屉式工具箱（见 §15）。
- **个人位置标记带方向箭头**，随设备朝向旋转。
- **点击地图空白处取经纬度**：落点十字 + 底部读数条；坐标格式（度/度分秒/UTM）在设置里切换。
- 两步路把「记录」「导航」并排放底部，我们沿用。
