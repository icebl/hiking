# MVP 开发任务清单（iOS）

> 版本：v0.1 · 2026-06-24
> 依据：PRD + 四大模块细化 + 技术选型 + UI 线框/视觉稿。
> 技术栈：Swift + SwiftUI / MapLibre Native / GRDB(SQLite) / CoreGPX / Core Location + CoreMotion / PMTiles。
> 标注：**P0=MVP 必须**，P1=二期，P2=后续。任务按依赖顺序分阶段（Sprint）。

---

## 总览：依赖关系

```
S0 工程地基 ─┬─ S1 数据层 ───┬─ S3 轨迹记录 ──┐
             └─ S2 地图/离线 ─┴─ S4 沿轨迹导航 ┼─ S6 联调打磨
                              └─ S5 导入/导出  ┘
```
原则：**先地基与数据，再定位与地图两条并行线，最后汇到导航。**

---

## S0 · 工程地基（P0）

- [ ] 0.1 建 iOS 工程（XcodeGen `project.yml` → `.xcodeproj`），SwiftUI App 生命周期。
- [ ] 0.2 接入 SPM 依赖：MapLibre、GRDB、CoreGPX。
- [ ] 0.3 Info.plist：定位权限文案（When/Always）、`UIBackgroundModes=location`、`NSMotionUsageDescription`。
- [ ] 0.4 目录结构与分层（App / Core / Features）。
- [ ] 0.5 四 Tab 骨架（首页/地图/轨迹/我的）+ 地图全屏隐藏 Tab 的导航容器。
- [ ] 0.6 设计 Token 落地（颜色/字体/圆角/间距 → SwiftUI 常量）。

## S1 · 数据层（P0，云同步预留）

- [ ] 1.1 GRDB 数据库初始化 + 迁移框架（`AppDatabase`）。
- [ ] 1.2 模型：`Track / TrackSegment? / TrackPoint / Waypoint / RecordingSession`。
- [ ] 1.3 **同步预留字段**：UUID 主键、createdAt/updatedAt、isDeleted（软删除）、isSynced。
- [ ] 1.4 `TrackRepository`：增删改查、列表、统计聚合。
- [ ] 1.5 轨迹点高效存储（批量写入、按轨迹分页查询）。

## S2 · 地图与离线（P0）

- [ ] 2.1 `MapLibreView`（UIViewRepresentable 包装 MLNMapView），WGS-84。
- [ ] 2.2 自带矢量底图 style（PMTiles 协议接入）。
- [ ] 2.3 地图控件：缩放 +/- + 滑块、定位、居中、公里标。
- [ ] 2.4 在地图上绘制轨迹折线 / 起终点 / 当前位置（带朝向箭头）。
- [ ] 2.5 等高线图层叠加 + 开关。
- [ ] 2.6 图层切换（自带 + 在线卫星，T_Google 离线）。
- [ ] 2.7 离线包下载：框选区域 + 选级别(≤z16) + 体积预估 + 下载/管理。
- [ ] 2.8 点击地图取经纬度（坐标格式 度/度分秒/UTM）。
- [ ] 2.9 数据生产管线（电脑侧）：planetiler 底图 + GDAL/tippecanoe 等高线 → PMTiles。

## S3 · 轨迹记录（P0，生死线）

- [ ] 3.1 `LocationManager`：Core Location，前台定位 + 精度/距离过滤。
- [ ] 3.2 **后台连续定位**：Background Modes + `allowsBackgroundLocationUpdates` + Always 权限引导。
- [ ] 3.3 采样策略：5s + 位移 5m；静止降频。
- [ ] 3.4 `AltimeterManager`：CoreMotion 气压计海拔（可用时优先，否则 GPS）。
- [ ] 3.5 实时统计：距离/用时（运动·全程）/爬升·下降（去噪阈值 5m）/海拔/速度。
- [ ] 3.6 去噪：低精度点丢弃、最小位移过滤、海拔平滑。
- [ ] 3.7 记录状态机：开始/暂停/继续/结束 + 自动暂停。
- [ ] 3.8 **崩溃恢复**：RecordingSession 增量落盘，启动检测未完成会话并恢复。
- [ ] 3.9 无信号断段：虚线"推测段"，不计入真实里程。
- [ ] 3.10 结束结算 → 生成 Track + 缩略图入库。
- [ ] 3.11 记录中 UI：实时数据面板 + 暂停/结束。

## S4 · 沿轨迹导航（P0）

- [ ] 4.1 `NavigationEngine`：当前点投影到计划线、沿线已行进里程、剩余距离/爬升。
- [ ] 4.2 预计算：计划线逐点累计距离/爬升缓存。
- [ ] 4.3 **偏航检测**：投影距离 + 防抖（持续 Ns）+ 滞回（25m/15m）+ 自交单调窗口。
- [ ] 4.4 偏航提醒：通知 + 震动；可选语音播报（间隔 5/10 分钟）；后台可触发。
- [ ] 4.5 方向选择（正/反向）+ 进入前同时记录开关。
- [ ] 4.6 定位差暂停判定、起点远未上线、到达终点仅提示。
- [ ] 4.7 导航 UI：计划线/已走/实走、状态条、偏航横幅、剩余信息。

## S5 · 导入 / 导出 / 分享（P0）

- [ ] 5.1 `GPXService`：GPX 1.1 解析（CoreGPX），trk/trkseg/trkpt/wpt/ele/time。
- [ ] 5.2 导入容错：缺 ele（DEM 补）/缺 time/编码/超大文件分段/多 trk 全导入/损坏不崩。
- [ ] 5.3 导入入口：系统"用本应用打开" + 分享面板接收 + APP 内文件选择。
- [ ] 5.4 导入预览页 → 确认入库。
- [ ] 5.5 导出 GPX（含 wpt）→ 系统分享面板（《导出轨迹文件》页，MVP 仅 GPX）。
- [ ] 5.6 海拔剖面图（轨迹详情，点击联动地图）。

## S6 · 联调与打磨（P0）

- [ ] 6.1 首页：累计数据卡 + 快捷入口（活动公告区 = 联网模块，见 P1）。
- [ ] 6.2 轨迹库列表 + 详情（地图/详情页签）+ 操作菜单。
- [ ] 6.3 设置页：采样/偏航阈值/语音/海拔来源/坐标格式/同时记录。
- [ ] 6.4 全程耗电与后台稳定性实测（4–8 小时）。
- [ ] 6.5 互通性测试：导出文件被两步路/六只脚/佳明正确读取，反之亦然。
- [ ] 6.6 强光可读性、空状态、错误与权限引导。
